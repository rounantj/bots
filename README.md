## Installation

PAT requires [Node.js](https://nodejs.org/) v12.20.0+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd src
npm i
node run dev
```

For production environments...

```sh
cd src
npm i
node run prod
```
